# Setup Docker
## 🧩 BƯỚC 1: Kiểm tra máy
**👉 Máy bạn cần:**
* Windows 10/11 64-bit
* Bật Virtualization
**Check nhanh:**
* Mở Task Manager → Performance → CPU
* Thấy dòng Virtualization: Enabled là OK ✅
Nếu Disabled → vào BIOS bật (Intel VT-x / SVM).
## 🧩 BƯỚC 2: Cài WSL 2 (quan trọng)
**Mở PowerShell (Run as Admin) rồi gõ:**
```powershell
wsl --install
``
* Máy sẽ tự cài:
	* WSL
	* Ubuntu
* Xong → restart máy 🔄
```
**👉 Check lại:**
```powershell
wsl --status
```
## 🧩 BƯỚC 3: Cài Docker Desktop
1. Vào trang: docker.com → Download Docker Desktop
2. Tải bản Docker Desktop for Windows
3. Cài như app bình thường
* Tick:
	* -[x] Use WSL 2
	* -[x] Add shortcut
👉 Cài xong → restart máy (nếu nó bảo)
## 🧩 BƯỚC 4: Mở Docker Desktop
* Mở Docker Desktop
* Đợi icon 🐳 chạy xong (không đỏ là được)
## 🧩 BƯỚC 5: Test Docker đã chạy chưa
Mở CMD / PowerShell:
```powershell
docker --version
docker run hello-world
```
Nếu thấy dòng:
> Hello from Docker! 🎉
